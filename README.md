# CofreLocal

## Descrição
O **CofreLocal** é um sistema de login **offline** criado para a atividade de Interação Humano–Computador (IHC).  
O objetivo é demonstrar um fluxo de autenticação **local** (sem internet e sem banco de dados), com **validação de dados** e **feedback claro ao usuário**.

**Tecnologias:** Python 3 + Tkinter (interface) + JSON local (simulação de usuários).

---

## Instruções de instalação
### Pré-requisitos
- Python 3 instalado
- Tkinter (normalmente já vem com o Python; em algumas distros Linux pode exigir pacote `python3-tk`)

1. Baixe/clonar o repositório
